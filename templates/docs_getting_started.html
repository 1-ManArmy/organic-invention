Test your setup with a simple API call to create your first AI agent.

Installation & Setup

Choose your preferred programming language and get started with our SDKs.

Python SDK

Most popular choice

# Install the SDK
pip install aidigitalfriend

# Basic usage
from
aidigitalfriend
import
Client

client = Client(api_key="your_key")
agent = client.agents.create(
name="My Agent"
)
