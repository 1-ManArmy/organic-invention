Chat Completions

Generate conversational responses using AI models

POST /chat/completions

Request Body

Parameter
Type
Required
Description
message
string
✓
The user's message to send to the AI
model
string
✓
AI model to use (e.g., "ai-friend-pro")
temperature
number
Randomness (0.0-2.0, default: 0.7)
max_tokens
integer
Maximum response length (default: 150)
stream
boolean
Stream response in real-time (default: false)

Example Request

cURL 
curl -X POST https://api.intelcoreai.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "What are the benefits of AI in healthcare?",
    "model": "ai-friend-pro",
    "temperature": 0.7,
    "max_tokens": 200
  }'

Example Response

{
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "created": 1690814400,
  "model": "ai-friend-pro",
  "choices": [{
    "index": 0,
    "message": {
      "role": "assistant",
      "content": "AI offers significant benefits in healthcare including improved diagnostic accuracy, personalized treatment plans, drug discovery acceleration, and administrative efficiency. It can analyze medical images, predict patient outcomes, and assist doctors in making more informed decisions."
    },
    "finish_reason": "stop"
  }],
  "usage": {
    "prompt_tokens": 12,
    "completion_tokens": 48,
    "total_tokens": 60
  }
}

AI Agents

Manage and interact with specialized AI agents

GET /agents
List all available AI agents
POST /agents
Create a new custom AI agent
GET /agents/{agent_id}
Get details of a specific AI agent
POST /agents/{agent_id}/chat

Chat with Agent

Send a message to a specific AI agent

Example Request 
curl -X POST https://api.intelcoreai.com/v1/agents/healthcare-bot/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "What should I know about diabetes management?",
    "context": {
      "patient_age": 45,
      "medical_history": ["hypertension"]
    }
  }'
