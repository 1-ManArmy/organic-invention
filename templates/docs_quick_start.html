Quick Start Guide

Get your AI Digital Friend up and running in under 5 minutes with our rapid deployment guide.

5 Minutes
Instant Results
Production Ready

3 Simple Steps

Follow these steps to deploy your first AI agent in minutes.

1

Install & Setup

One command installation

# Install AI Digital Friend
npm install -g @aidf/cli

# Or using pip
pip install aidigitalfriend

Install the AI Digital Friend CLI or Python package with a single command. No complex setup required.

2

Configure API Key

Secure authentication

# Set your API key
aidf config set-key YOUR_API_KEY

# Verify authentication
aidf auth verify

Get your free API key from the dashboard and configure it with one command. Start with generous free limits.

3

Deploy & Test

Launch your first agent

# Create and deploy agent
aidf create chatbot
aidf deploy --name my-agent

# Test your agent
aidf chat "Hello, world!"

Create your first AI agent with templates and deploy instantly. Test it immediately with the built-in chat interface.
