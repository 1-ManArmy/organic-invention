Most common cause: Invalid or missing API key in your requests.

Quick Fix:

  • Verify your API key is correct
  • Check the Authorization header format

Rate Limit Exceeded

API Issues

Request and response problems

400 Bad Request: Invalid JSON

Problem:

Your request contains malformed JSON or missing required fields.

Common Causes:
  • Missing quotes around string values
  • Trailing commas in JSON objects
  • Missing required fields
  • Incorrect data types
Solution:
// ❌ Wrong
{
  name: "My Agent",
  type: "chat",
}

// ✅ Correct
{
  "name": "My Agent",
  "type": "chat"
}

Pro Tip: Use a JSON validator before sending requests.

Request Timeout Issues

Problem:

Requests are timing out before receiving a response from the API.

Common Causes:
  • Complex AI processing taking too long
  • Network connectivity issues
  • Default timeout too short
  • Server overload
Solution:
# Python example
import
requests


response = requests.post(
  url,
  json=data,
  timeout=30
# Increase timeout
)

Recommendation: Set timeout to 30-60 seconds for AI requests.
