API-Specific Error Codes

Custom error codes specific to our AI platform and their resolution steps.

AI Agent Errors (1000-1099)

1001 - Agent Not Found
The specified AI agent does not exist or has been deleted.
1002 - Agent Busy
The AI agent is currently processing another request.
1003 - Agent Disabled
The AI agent has been disabled by the owner.

Authentication Errors (2000-2099)

2001 - Invalid API Key
The provided API key is not valid or has expired.
2002 - Token Expired
The authentication token has expired and needs renewal.
2003 - Insufficient Permissions
Your account doesn't have permission for this operation.

Rate Limit Errors (3000-3099)

3001 - Rate Limit Exceeded
You have exceeded your API rate limit. Wait before retrying.
3002 - Quota Exceeded
Monthly API usage quota has been exceeded.
3003 - Concurrent Limit
Too many concurrent requests. Reduce parallel calls.

Data Processing Errors (4000-4099)

4001 - Invalid Input Format
The input data format is not supported or malformed.
4002 - File Too Large
The uploaded file exceeds the maximum size limit.
4003 - Processing Timeout
The request timed out during processing.

Error Resolution Tips

Check Error Details

Always read the error message and check the error_details field for specific information.

Implement Retry Logic

For temporary errors (5xx), implement exponential backoff retry logic.

Log Errors

Log error responses with request IDs for easier debugging and support.

Contact Support

If errors persist, contact our support team with the request ID and error details.
