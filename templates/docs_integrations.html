Quick Start Integration

Get started with popular integrations in just a few steps.

Slack Integration

Add AI agents to your Slack workspace

Setup Steps:

1
Create a Slack app in your workspace
2
Configure webhook URL in AI Digital Friend
3
Install the app and start chatting with your AI agent
# Install via pip
pip install aidigitalfriend[slack]

# Configure integration
from
aidigitalfriend.integrations
import
SlackBot

bot = SlackBot(
api_key="your_api_key",
slack_token="xoxb-your-token"
)
bot.start()

Discord Integration

Deploy AI bots to Discord servers

Setup Steps:

1
Create Discord application and bot
2
Set up bot permissions and intents
3
Invite bot to server and configure commands
# Install Discord integration
pip install aidigitalfriend[discord]

# Create Discord bot
from
aidigitalfriend.integrations
import
DiscordBot

bot = DiscordBot(
api_key="your_api_key",
discord_token="your_bot_token"
)
bot.run()
