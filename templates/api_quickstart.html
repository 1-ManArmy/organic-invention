Make Your First API Call

Let's send a simple chat message to our AI Digital Friend API using cURL:

Terminal 
curl -X POST https://api.intelcoreai.com/v1/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello! How can you help me today?",
    "model": "ai-friend-pro",
    "temperature": 0.7,
    "max_tokens": 150
  }'
cURL 
curl -X POST https://api.intelcoreai.com/v1/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello! How can you help me today?",
    "model": "ai-friend-pro",
    "temperature": 0.7
  }'
Python 
import requests

url = "https://api.intelcoreai.com/v1/chat"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "message": "Hello! How can you help me today?",
    "model": "ai-friend-pro",
    "temperature": 0.7
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
JavaScript (Browser) 
fetch('https://api.intelcoreai.com/v1/chat', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    message: 'Hello! How can you help me today?',
    model: 'ai-friend-pro',
    temperature: 0.7
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
Node.js 
const axios = require('axios');

const config = {
  method: 'post',
  url: 'https://api.intelcoreai.com/v1/chat',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  data: {
    message: 'Hello! How can you help me today?',
    model: 'ai-friend-pro',
    temperature: 0.7
  }
};

axios(config)
  .then(response => console.log(response.data))
  .catch(error => console.log(error));
3

Handle the Response

The API will return a JSON response containing the AI's message and additional metadata:

JSON Response 
{
  "id": "chatcmpl-7qNH8",
  "object": "chat.completion",
  "created": 1690814400,
  "model": "ai-friend-pro",
  "choices": [{
    "index": 0,
    "message": {
      "role": "assistant",
      "content": "Hello! I'm here to help you with a wide range of tasks. I can assist with answering questions, providing information, helping with analysis, creative writing, coding, and much more. What would you like to work on today?"
    },
    "finish_reason": "stop"
  }],
  "usage": {
    "prompt_tokens": 12,
    "completion_tokens": 45,
    "total_tokens": 57
  }
}

Response Fields Explained

  • id: Unique identifier for this chat completion
  • choices: Array containing the AI's response(s)
  • message.content: The actual text response from the AI
  • usage: Token usage statistics for billing and monitoring
