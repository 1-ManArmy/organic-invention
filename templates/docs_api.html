API Endpoints

Explore all available endpoints organized by functionality.

Agents

Create, manage, and interact with AI agents

Create Agent

POST
POST
/agents

Create a new AI agent with custom personality and capabilities.

Request Body:
{
  "name": "My AI Assistant",
  "personality": "helpful and friendly",
  "capabilities": ["chat", "analysis"]
}
Response:
{
  "id": "agent_12345",
  "name": "My AI Assistant",
  "status": "active",
  "created_at": "2024-01-15T10:00:00Z"
}
201 Created
400 Bad Request
401 Unauthorized

Get Agent

GET
GET
/agents/{agent_id}

Retrieve details of a specific agent by ID.

Path Parameters:
agent_id (string, required) - The unique identifier of the agent
200 OK
404 Not Found

List Agents

GET
GET
/agents?limit=10&offset=0

Retrieve a paginated list of all your agents.

Query Parameters:
limit (integer, optional) - Number of agents to return (default: 10, max: 100)
offset (integer, optional) - Number of agents to skip (default: 0)
200 OK
401 Unauthorized

Chat

Send messages and receive responses from agents

Send Message

POST
POST
/agents/{agent_id}/chat

Send a message to an agent and receive a response.

Request Body:
{
  "message": "Hello, how can you help me today?",
  "stream": false
}
200 OK
404 Agent Not Found
