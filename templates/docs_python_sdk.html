Quick Start

Get started with the Python SDK in just a few lines of code.

Installation 

# Install via pip
pip install aidf

# Or with conda
conda install -c conda-forge aidf

# Install development version
pip install git+https://github.com/aidf/python-sdk.git

Requires Python 3.8 or higher. Compatible with asyncio for asynchronous operations.

Authentication 

import aidf

# Set your API key
client = aidf.Client(api_key="your-api-key")

# Or use environment variable
# export AIDF_API_KEY="your-api-key"
client = aidf.Client()

Get your API key from the dashboard.

Basic Usage Examples

Common use cases and examples to get you started quickly.

Chat Agent

Create a conversational AI agent with just a few lines of code.

import aidf

# Create a client
client = aidf.Client(api_key="your-api-key")

# Create a chat agent
agent = client.agents.create(
    name="My Assistant",
    description="A helpful AI assistant",
    model="gpt-4",
    system_prompt="You are a helpful assistant."
)

# Start a conversation
response = agent.chat("Hello! How can you help me today?")
print(response.message)

# Continue the conversation
response = agent.chat("Tell me about Python programming")
print(response.message)

Streaming Responses

Stream responses for real-time user experiences.

# Stream chat responses
for chunk in agent.chat_stream("Write a story about AI"):
    print(chunk.content, end="")

# Async streaming
async def stream_chat():
    async for chunk in agent.achat_stream("Hello!"):
        print(chunk.content, end="")

# Run async function
import asyncio
asyncio.run(stream_chat())

Advanced Features

Memory Management

Persistent conversation memory and context management.

Custom Models

Use custom models and fine-tuned implementations.

)

Plugin System

Extend functionality with the plugin ecosystem.

Analytics & Monitoring
